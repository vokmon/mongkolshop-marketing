# Content Schedule — MongkolShop

## Weekly Template (Base Schedule)

4 editorial slots/day + game slot (ทุกวัน) + product slot แยกต่างหาก
**Content agents กำหนด `scheduled_publish_time` ตาม slot ของตัวเองตอนสร้าง content**

**Slot 1 = horoscope-agent · Slot 2 = news-agent (auto) · Slot 3 = story-agent · Slot 4 = life-topic-agent · Slot 5 = game-agent (daily)**
_Slot 3-5 — ตารางนี้เป็นแนวทาง rotation ปรับได้ตามบริบทวันนั้น (เทศกาล, กระแส, engagement)_

### Game Slot Rules (Controlled Random)

- ลงทุกวันเวลา 20:30
- สุ่มโหมดจาก 2 แบบ (`emoji_guess`, `syllable_hint_guess`) แต่ห้ามโหมดเดิมซ้ำติดกัน
- คำตอบห้ามซ้ำย้อนหลัง 30 วัน (ตรวจจาก `outputs/recent-log.json`)
- ความยากแบบ weighted random: easy 60% / medium 30% / hard 10%
- คำตอบต้อง 3-7 พยางค์เสมอ

**Product slot (แยก): 15:00 น. — 3-4 วัน/สัปดาห์ กระจายไม่ติดกัน**
วันพระ / วันศาสนา / เทศกาล → งด product post วันนั้น

## Engagement Baseline (ทุกโพสต์ต้องมีในตัว)

ทุก editorial post ในตารางนี้ต้องมี engagement เล็กๆ อยู่ในโพสต์นั้นเลย โดยไม่ต้องพึ่ง optimizer เพิ่ม:

- มี comment trigger สั้นๆ 1 จุดท้ายโพสต์ (เช่น คอมเมนต์วันเกิด / พิมพ์ 1-2-3)
- มี CTA เดียวชัดๆ 1 จุด
- hashtag 3-5 อัน
- caption อ่านง่ายบนมือถือ ไม่ยืดยาวเกินจำเป็น

แนวคิดคือให้ content agents แต่ละตัว (horoscope/story/life-topic/news) เขียนโพสต์ที่พร้อม engage ตั้งแต่ต้นทาง

## Posting Time Rationale

| เวลา      | พฤติกรรม User                                       |
| --------- | --------------------------------------------------- |
| 7:00 น.   | เช็คมือถือก่อนออกจากบ้าน — ดวงวันนี้ตรงกับ behavior |
| 8-9:00 น. | ระหว่างเดินทาง/เริ่มทำงาน — เช็คข่าวประจำวัน        |
| 12:00 น.  | พักกลางวัน — อ่านสาระ เรื่องเล่า                    |
| 19:00 น.  | หลังเลิกงาน — engage, ตัดสินใจซื้อ                  |
| 20:30 น.  | ช่วงพักก่อนนอน — เหมาะกับโพสต์เกม/คอมเมนต์เร็ว      |
