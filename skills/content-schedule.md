# Content Schedule — MongkolShop

## Weekly Template (Base Schedule)

4 editorial slots/day + product slot แยกต่างหาก
**Content agents กำหนด `scheduled_publish_time` ตาม slot ของตัวเองตอนสร้าง content**

**Slot 1 = horoscope-agent · Slot 2 = news-agent (auto) · Slot 3 = story-agent · Slot 4 = life-topic-agent**
_Slot 3, 4 — ตารางนี้เป็นแนวทาง rotation ปรับได้ตามบริบทวันนั้น (เทศกาล, กระแส)_

| ----------- | ------------- | ------------------ | ----------------------- | ---------------------- |

## Posting Time Rationale

| เวลา      | พฤติกรรม User                                       |
| --------- | --------------------------------------------------- |
| 7:00 น.   | เช็คมือถือก่อนออกจากบ้าน — ดวงวันนี้ตรงกับ behavior |
| 8-9:00 น. | ระหว่างเดินทาง/เริ่มทำงาน — เช็คข่าวประจำวัน        |
| 12:00 น.  | พักกลางวัน — อ่านสาระ เรื่องเล่า                    |
| 19:00 น.  | หลังเลิกงาน — engage, ตัดสินใจซื้อ                  |
