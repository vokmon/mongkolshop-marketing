# Hook Generator — MongkolArt

## หลักการ

- หยุดนิ้วได้ใน 3 วินาทีแรก
- ใช้ตัวเลข อารมณ์ตกใจ หรือ curiosity gap เสมอ
- เชื่อมกับ pain point: ดวงไม่ดี / อยากโชคดี / อยากรวย / กลัวเรื่องร้าย
- ทำให้ user รู้สึกว่าเกี่ยวกับตัวเขาโดยตรง
- ห้ามบอกทุกอย่างใน hook — ต้องมีช่องว่างให้อยากรู้ต่อ
- ความยาวไม่เกิน 10 คำ

## ตัวอย่าง Formats (ไม่จำกัดเฉพาะนี้)

สร้าง hook ในรูปแบบใหม่ได้เสมอ ถ้ามันหยุดนิ้วได้และเข้าหลักการข้างบน
ตัวอย่างด้านล่างเป็นแค่จุดเริ่มต้น ไม่ใช่ list ที่ตายตัว

### แบบ "เปิดเผย / ตกใจ"

[ตัวเลขหรือข้อเท็จจริง] + [สิ่งที่น่าตกใจ]

> "90% คนไทยบูชาเทพผิดองค์โดยไม่รู้ตัว"
> "รูปมงคลที่บ้านคุณอาจดูดพลังงานไปโดยไม่รู้ตัว"

### แบบ "เฉพาะกลุ่ม"

คนเกิด[วัน/ราศี/ปี] + [สิ่งที่ต้องรู้]

> "คนเกิดวันอังคารห้ามบูชาองค์นี้เด็ดขาด"
> "ราศีกุมภ์ปีนี้ดวงพุ่ง ถ้ามีรูปมงคลนี้"

### แบบ "ทดสอบ / เปรียบเทียบ"

[A] vs [B] + [ผลที่น่าสนใจ]

> "รูปมงคลทั่วไป vs รูปมงคลตรงดวง ต่างกันยังไง?"

### แบบ "before/after"

ก่อน/หลัง + [ผลลัพธ์ที่น่าเชื่อ]

> "30 วันหลังเปลี่ยนรูปมงคล ชีวิตเปลี่ยนยังไง"

### แบบ "คำถามปลุกอารมณ์"

[ปัญหา/ความกลัว] + คำถาม

> "ทำไมอธิษฐานแล้วไม่สมหวังสักที?"
> "รูปมงคลที่ตั้งอยู่ช่วยคุณได้จริงไหม?"

## Video Hook Overlay (caption บน scene 1)

ใช้สำหรับ hook ที่แสดงเป็น text overlay บน video — ต่างจาก caption post ตรงที่ต้องสั้นกว่า อ่านได้ใน 3 วินาที และ ห้าม over-claim

ตัวอย่างด้านล่างเป็นแค่จุดเริ่มต้น — สร้างแบบใหม่ได้เสมอตาม angle ของ content

### ตั้งคำถามชวนคิด

```
เคยรู้สึกไหม
ว่าบางรูปดูแล้วใจนิ่งขึ้น?
```

```
วันที่รู้สึกหนัก
คุณหยิบมือถือขึ้นมาทำอะไรเป็นอย่างแรก?
```

### ชวนสังเกต

```
wallpaper มือถือของคุณ
ตอนนี้เป็นอะไรอยู่?
```

```
คนที่ดวงดี
มักมีอะไรบางอย่างที่คนอื่นมองข้าม
```

### เล่าเรื่องจากประสบการณ์คนอื่น

```
เพื่อนบอกว่าตั้งรูปนี้แล้ว
รู้สึกมั่นใจขึ้นทุกวัน
```

```
มีคนบอกว่า
วันที่เครียดที่สุด รูปนี้ช่วยได้
```

```
รูปมงคลที่ใช่
ไม่ได้เหมือนกันทุกคน
```

## สิ่งที่ต้องหลีกเลี่ยง

- อย่าใช้ภาษาทางการเกินไป
- อย่า overpromise ("รวยแน่นอน 100%")
- อย่าเริ่มด้วย "สวัสดีครับ/ค่ะ"
