# MongkolShop Marketing

Marketing automation and content generation for MongkolArt — น้องมงคล LINE OA Chatbot (`@652hgnwz`).

Goal: drive sales to the LINE OA channel through automated content creation, image generation, and social media posting.

## Scope

- **Content generation** — captions, posts, hashtags for Facebook and LINE
- **Image generation** — promotional visuals using AI image tools
- **Posting automation** — scheduled and triggered posts to Facebook Page and LINE Broadcast
- **Campaign planning** — strategy, content calendar, and briefs

## Channels

| Channel | Purpose |
|---|---|
| Facebook Page | Organic content, promotions, retargeting |
| LINE OA `@652hgnwz` | Broadcast to existing customers |

## Tools

### AI CLI Tools
AI CLI tools (e.g. `codex`, `claude`, `gemini`) are available for content generation and automation.

### No-code Automation
- **n8n** — workflow automation for scheduling and posting
- **Facebook Graph API** — publish posts and manage Facebook Page
- **LINE Messaging API** — broadcast messages to LINE OA followers

## Product Context

This project drives traffic to the MongkolArt LINE OA chatbot. The product:
- Collects 7 fields from users via natural conversation
- Takes payment via Stripe (159 THB)
- Generates a personalized sacred art wallpaper using AI
- Delivers via LINE push message

Target audience: Thai users interested in sacred art, deities, and auspicious symbols across Hindu, Chinese, and Thai traditions.
